## Get started in 3 steps

<Steps>
  <Step title="Create a project and get your API key">
    Sign up at [lumiqtrace.com](https://lumiqtrace.com), create an organization, and create your first project. Your project API key (`lqt_...`) is shown once on creation — copy it now.
  </Step>
  <Step title="Install the SDK and wrap your LLM client">
    Install `@lumiqtrace/sdk` (Node.js) or `lumiqtrace` (Python), initialize with your API key, and wrap your existing LLM client. No changes to your business logic required.

    ```typescript
    import { lumiqtrace } from "@lumiqtrace/sdk";
    import OpenAI from "openai";

    lumiqtrace.init({ apiKey: "lqt_your_key_here" });
    const openai = lumiqtrace.wrapOpenAI(new OpenAI());
    ```
  </Step>
  <Step title="View traces in the dashboard">
    Make an LLM call as usual. Within seconds, the trace appears in your LumiqTrace dashboard with full cost, latency, and token breakdowns.
  </Step>
</Steps>

<Note>
  LumiqTrace is designed to be zero-latency for your application. All telemetry is batched and sent asynchronously — your LLM calls are never blocked.
</Note>
